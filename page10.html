How can assistive technologies help?

Navigating digital content with disabilities can be challenging. But there are technologies to help with that.

  1. Screen Reader

    Helps people to perceive content of a website through conversion to other formats such as speech or braille.

  2. Braille Display

    Computer output device for blind people that displays characters in braille.

  3. Voice Control

    Input and output is done via spoken language.

  4. Sip-and-Puff Switches

    This switch translates sips and puffs into independent switch closures.

  5. Screen Magnifier

    Helps people to see content of a web page.
